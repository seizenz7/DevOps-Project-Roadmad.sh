# Server Performance Stats
Write a script to analyse basic server performance stats.

## Project URL
https://roadmap.sh/projects/server-stats

## Project Details
Goal of this project is to write a script to analyse server performance stats.

### Requirements

Write a script server-stats.sh that can analyse basic server performance stats. The script should be able to run on any Linux server and it should give the following stats:

- Total CPU usage

- Total memory usage (Free vs Used including percentage)

- Total disk usage (Free vs Used including percentage)

- Top 5 processes by CPU usage

- Top 5 processes by memory usage
